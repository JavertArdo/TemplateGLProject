# TemplateGLProject
Simple CMake OpenGL Template Project
* C++ 11
* CMake 3.11
* GLFW Library
* GLUT Library

## Dependencies
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes

### Ubuntu
```
freeglut3-dev xorg-dev
```

### Arch Linux
```
libglvnd
```
